# 用户手册

## 系统功能介绍

### 显示系统
- **四位数码管**：显示总计数（0000-9999）
- **第一个两位数**：显示当前批次计数（00-99）
- **第二个两位数**：显示批次目标值（默认05）

### 状态指示
- **LED1 (P3.6)**：系统状态指示（快闪=计数）
- **LED2 (P3.7)**：电机状态指示（亮=运行，灭=停止）

### 用户交互
- **按钮 (P3.2)**：模拟传感器输入，每次按下计数+1
- **复位电路**：拔插RST到GND的导线实现系统复位

## 操作流程
1. 连接所有硬件并上电
2. 观察系统启动自检（LED双闪）
3. 按下按钮进行计数测试
4. 观察数码管显示更新
5. 当批次计数达到目标值时，电机会自动运行

### 参数调整
修改 `firmware/main.c` 中的参数：
unsigned char target_count = 10;  // 改为每10个一批
DelayMS(3000);                    // 电机运行时间改为3秒
